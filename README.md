# HiPeople Web 🌐

The **HiPeople Web** application is the browser-based platform that allows users to access HiPeople without installing a mobile app. It provides the same core functionality with a responsive and modern web interface.

---

## 🔍 Overview

This repository contains the **frontend web application** for HiPeople. It connects with the HiPeople API to deliver a fast, interactive, and accessible experience across all modern browsers.

---

## ✨ Key Features

### 🧭 Discover & Connect

* Find people nearby
* Explore local communities and groups

### 👤 Profile Management

* Create and update profiles
* Manage privacy and preferences

### 💬 Messaging

* Real-time chat via web
* Group conversations

### 🗺️ Location-Based UI

* Map-based discovery
* Distance indicators

### 📱 Responsive Design

* Works on desktop, tablet, and mobile browsers

---

## 🛠️ Technology Stack

* **Framework:** React.js / Next.js
* **Styling:** Tailwind CSS / CSS Modules
* **State Management:** Context API / Redux
* **API Integration:** REST APIs
* **Real-Time:** Socket.IO

---

## 📦 Project Structure (Example)

* `pages/` or `app/` – Routes
* `components/` – UI components
* `styles/` – Styling files
* `services/` – API calls
